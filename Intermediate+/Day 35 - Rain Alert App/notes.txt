--> What is API authentication?
We're going to learn how to authenticate ourselves with an API provider so that we can access more valuable and secure data.

APIs are a way of selling data, as it is very hard to maintain such data.
But generally, all API providers offer a free tier where users can test out the functionality.
To avoid misuse of this free tier, they use API keys similar to personal account number and password.
